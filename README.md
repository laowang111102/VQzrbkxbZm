# 项目简介 python334

本项目是基于 Python 和协同过滤算法的商品推荐系统设计与实现。系统主要包括以下功能模块，并使用 Django 框架和 VUE 技术进行前后端开发。

## 技术栈

- **后端技术**：Python、Django 框架
- **前端技术**：VUE
- **开发工具**：PyCharm

## 功能模块

- **系统首页**：展示系统整体信息
- **个人中心**：用户可以管理自己的信息
- **用户管理**：管理员管理用户账号
- **店铺信息管理**：商家管理店铺信息
- **商品信息管理**：管理商品的详细信息
- **商品分类管理**：对商品进行分类管理
- **系统管理**：进行系统设置和维护
- **订单管理**：处理订单相关信息

## 系统角色

- **管理员**：负责系统的整体管理和用户账号管理
- **商家**：管理店铺和商品信息
- **普通用户**：使用推荐系统进行商品浏览和购买

## 运行环境

- Python 环境推荐使用 Python 3.x
- 数据库依赖请参考 `requirements.txt`
- 前端 Node.js 环境要求请参考前端项目文档

## 部署步骤

1. 确保后端所需 Python 环境和第三方库安装完毕
2. 配置数据库连接信息
3. 运行后端 Django 项目
4. 构建并运行前端 VUE 项目

## 目录结构

```
项目根目录/

├── frontend/              # 前端 VUE 项目目录

```
